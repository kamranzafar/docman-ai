# Docman AI
A reference implementation of AI enabled document management system.

## Technology Stack
- Spring Boot for API
- MINIO (Object store)
- MongoDB (Metadata store)
- Open Search for document indexing
- Ollama AI for text extraction
- Temporal for document workflow execution
- Kafka to emit document workflow events

## AI Models tested
- llava (PDFs and Images)
- llava:3b (PDFs and Images)
- llama3 (Text files only)
